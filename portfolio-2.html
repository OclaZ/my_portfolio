CleanMadina

Overview

CleanMadina is a Moroccan waste management application developed using Next.js. This app aims to streamline waste collection processes and enhance community engagement in environmental sustainability.

Features

  • User-Friendly Interface: Intuitive design for easy navigation and user experience.
  • Waste Reporting: Users can report waste issues within their community.
  • Rewards System: Users earn rewards for participating in waste management initiatives.
  • Notifications: Timely updates on waste collection schedules and community activities.

Technologies Used

  • Frontend: Next.js
  • Database: PostgreSQL
  • ORM: Drizzle ORM
  • Deployment: Vercel

Medical RAG Knowledge Base

Overview

Medical RAG Knowledge Base is a powerful healthcare document retrieval system built with Next.js and Pinecone. This app enables healthcare professionals to efficiently store, manage, and access medical documents, creating a streamlined workflow for medical research and knowledge retrieval.

Features

  • User-Friendly Interface: Intuitive design for easy navigation and user experience.
  • Document Storage: Securely store and manage healthcare-related documents.
  • Advanced Search: Efficient retrieval of documents using Pinecone’s vector search capabilities.
  • Version Control: Track document updates with version control for medical research papers.

Technologies Used

  • Frontend: Next.js
  • Database: Pinecone
  • Deployment: Vercel
